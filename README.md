## 👩‍💻 About Me

I am a **Software Engineering student** focused on building practical, scalable, and maintainable software solutions.

My main focus is **backend development**, especially with **C#, .NET Core, PHP, Laravel, SQL, RESTful APIs, authentication, database operations, and layered architecture**.

I also have hands-on **iOS development** experience with **Swift, UIKit, SwiftUI, Firebase, Moya, Alamofire, SwiftData, MapKit, and OpenAI API integrations**, which helps me understand software products from backend services to user-facing applications.

I enjoy learning by building, improving step by step, and turning technical knowledge into real projects.

## 🧭 Focus & Experience

- Building backend services with **C#, .NET Core, PHP, Laravel, and RESTful APIs**
- Working with **SQL, Microsoft SQL Server, Oracle Database, and PL/SQL**
- Developing authentication and secure API structures with **JWT**
- Applying software architecture principles such as **Clean Architecture, Onion Architecture, MVC, Repository Pattern, and MVVM**
- Building iOS applications using **Swift, UIKit, SwiftUI, Firebase, Moya, Alamofire, SwiftData, MapKit, and CoreLocation**
- Connecting mobile applications with backend services and API-driven systems
- Exploring AI-powered software features with **OpenAI APIs**
- Improving through internships, bootcamps, technical projects, and hands-on practice
